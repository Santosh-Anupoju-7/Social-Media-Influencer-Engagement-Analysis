# 📊 Social Media Influencer Engagement Analysis

## 📌 Project Overview

This project analyzes social media influencer data to understand engagement patterns, audience interaction, and performance metrics.

## 🎯 Objectives

* Identify top-performing influencers
* Analyze engagement rates
* Compare follower count vs engagement
* Visualize trends using charts

## 📂 Dataset

The dataset includes:

* Influencer name
* Followers
* Engagement rate
* Likes, comments, and other metrics

## 🛠️ Tools Used

* Python 🐍
* Pandas
* Matplotlib / Seaborn
* Google Colab

## 📈 Key Insights

* Influencers with moderate followers often have higher engagement
* Engagement rate is not directly proportional to follower count
* Some niches perform better than others
